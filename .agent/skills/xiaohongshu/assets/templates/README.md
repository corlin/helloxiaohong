# 内容模板目录

本目录用于存放小红书内容模板，帮助快速创建标准化的内容。

## 模板类型

### 1. 美食模板
```
标题：{美食名称}制作教程
内容：今天分享一道{美食名称}的制作方法，简单易学，味道鲜美。
食材：{食材列表}
步骤：{制作步骤}
小贴士：{制作技巧}
🚩素材来自：xiaohongshu-mcp
标签：#美食 #教程 #家常菜
```

### 2. 旅行模板
```
标题：{地点}旅行攻略
内容：这次去了{地点}，分享一下我的旅行经验和攻略。
交通：{交通方式}
住宿：{住宿推荐}
景点：{景点介绍}
美食：{当地美食}
🚩素材来自：xiaohongshu-mcp
标签：#旅行 #攻略 #景点
```

### 3. 生活模板
```
标题：{生活主题}分享
内容：关于{生活主题}的一些心得和体会。
个人感受：{感受描述}
实用技巧：{技巧分享}
推荐物品：{物品推荐}
🚩素材来自：xiaohongshu-mcp
标签：#生活 #分享 #日常
```

## 使用方法

1. 选择合适的模板
2. 替换占位符内容
3. 根据需要调整结构
4. 添加相关图片和标签

## 自定义模板

可以根据自己的需求创建新的模板，保持以下原则：
- 内容结构清晰
- 语言简洁明了
- 符合平台规范
- 便于快速修改
